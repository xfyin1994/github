# github使用操作

# 1.标题
```
在文字前写#,注意文字与#之间有一个空格

# 一级标题
## 二级标题
### 三级标题
以此类推 

或者用连续的减号或等号写在文字之下：

标题
---
```


# 2.粗体斜体
```
**这个是粗体**
*这个是斜体*
***这个是粗体加斜体***
```
